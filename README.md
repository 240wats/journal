# Journal

Este es un proyecto web estático con un diseño editorial de 3 columnas.

## Estructura
- `index.html`: Archivo principal con la estructura y estilos (CSS) del proyecto.
- Imágenes (`.jpg`, `.JPG`): Archivos multimedia que componen el diseño de la galería.

## Despliegue en GitHub Pages
Para visualizar esta web online gratuitamente (similar a Netlify):
1. Subí todos los archivos a un repositorio en GitHub.
2. Ve a la pestaña **Settings** (Configuración) del repositorio.
3. En el menú izquierdo, busca **Pages**.
4. Bajo *Build and deployment*, en la opción *Source*, seleccioná **Deploy from a branch**.
5. En *Branch*, elegí `main` (o `master`) y guardá.
6. ¡En unos minutos, tu página estará online!
